# 방명록 홈서버 구축 과정

1. 도메인 구매 및 네임서버를 cloudflare 로 두고 홈서버에 tunnel 설정
2. FastAPI + SQLite 방명록 서버
3. github action self hosted runner 홈서버로 배포 자동화
4. 로컬에서도 테스트 가능하도록 설정
5. 원격작업 위해 cloudflare ssh 터널 설정
6. 블로그에 방명록/댓글 적용
7. [git actions 마켓플레이스](https://github.com/marketplace/actions/ga-stats-fetcher)에 GA 정보를 입력받아 JSON 파일을 생성해주는 GitHub Action 배포
8. git actions 에서 GA 통계정보를 주기적으로 받아오는 자동화 설정
