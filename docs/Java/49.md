간단하게 FastAPI + SQLite + Cloudflare Tunnel 로 홈서버에 방명록 서버를 만들어서 적용한 과정을 정리합니다.

# 방명록 홈서버 구축 과정

1. 도메인 구매 및 네임서버 cloudflare 로 두고 홈서버에 tunnel 설정
2. FastAPI + SQLite 방명록 서버
3. github action self hosted runner 홈서버로 배포 자동화
4. 로컬에서도 테스트 가능하도록 설정
5. 원격작업 위해 cloudflare ssh 터널 설정
6. 블로그에 방명록/댓글 적용
