Reclaim Policy 는 PersistentVolume 을 삭제할 때 어떻게 할 지 정하는 정책.
* Retain : 삭제해도 PV 은 삭제되지 않고 남아있음.
* Recycle : 삭제하면 PV 을 삭제하고 디렉토리를 비움.
* Delete : 삭제하면 PV 을 삭제하고 디렉토리도 삭제함.
